A Django-based backend for a travel listings platform.  
This milestone focuses on project setup, MySQL integration, Swagger documentation, and CORS configuration.

## Features

- Django + DRF setup
- PostgraySQL database integration
- Swagger auto-documentation at `/swagger/`
- Secure `.env` environment handling using `django-environ`
- REST API ready structure

## Installation

```bash
git clone https://github.com/<your-username>/alx_travel_app.git
cd alx_travel_app
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
